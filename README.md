## Just Run It Already.... 🥱

It is assumed node and angular is installed as global dependencies.  

From the root of the project:

    $ docker compose up -d
     
Run the server:

    $ cd backend
    $ npm run start

In a new terminal tab:

    $ cd frontend
    $ ng o -s
